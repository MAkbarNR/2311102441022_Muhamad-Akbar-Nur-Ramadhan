## Cara Menjalankan Project

1. Salin dulu boss

```shell
git clone https://github.com/akhmadqasim/portfolio.git
```

2. Buat vir en boss

```shell
# Universal Python
python -m venv .venv
```
3. Activate kan .venv boss

```shell
# Windows
.venv\Scripts\activate
```

4. Terakhir install Django boss

```shell
pip install django
```
